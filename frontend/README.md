# Front End Built with: 

- [NextJS]
- [TailwindCSS]
- [Jest]
- [Axios]

## Requirements

Frontend:

To run dev server: 
```bash
bun run dev
```

To lint roll code:
```bash
npx prettier . --write
npm run lint
```

To run tests:
```bash
npx jest
```

Open [http://localhost:3000](http://localhost:3000) with your browser.
